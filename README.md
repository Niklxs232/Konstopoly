# Konstopoly

[![Build](https://github.com/Niklxs232/Konstopoly/actions/workflows/scala.yml/badge.svg)](https://github.com/Niklxs232/Konstopoly/actions/workflows/scala.yml)
[![Coverage Status](https://coveralls.io/repos/github/Niklxs232/Konstopoly/badge.svg?branch=master)](https://coveralls.io/github/Niklxs232/Konstopoly?branch=master)

Ein Monopoly-Brettspiel in Konstanz, gebaut als HTWG Software Engineering Projekt.

## Build & Run

```bash
sbt compile          # Kompilieren
sbt test             # Tests ausfuehren
sbt run              # Spiel starten
sbt coverage test coverageReport   # Coverage-Report generieren
sbt stryker          # Mutation Testing ausfuehren
```
